## Getting Started

### Prerequisites
```bash
rust >= 1.68.0
python >= 3.9
cuda >= 11.0
postgresql >= 14
solidity >= 0.8.0
```

### Installation
1. Clone the repository
```bash
git clone https://github.com/your-org/qgets.git
cd qgets
```

2. Install dependencies
```bash
cargo install --path .
pip install -r requirements.txt
```

3. Initialize the database
```bash
./scripts/init-db.sh
```

4. Start the simulation platform
```bash
cargo run --release
```
